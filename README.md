# Surugaya Vue 專案

這是一個使用 Vue 3 + Vite 建立的專案。

## 專案設置

```sh
npm install
```

### 編譯和熱重載用於開發

```sh
npm run dev
```

### 編譯和最小化用於生產

```sh
npm run build
```

### 預覽生產構建

```sh
npm run preview
```

## 使用技術

- Vue 3
- Vite
- JavaScript

## 開始使用

1. 確保已安裝 Node.js (建議版本 16 或以上)
2. 執行 `npm install` 安裝相依套件
3. 執行 `npm run dev` 開始開發伺服器
4. 在瀏覽器中開啟 `http://localhost:5173`

## 專案結構

```
├── public/          # 靜態資源
├── src/
│   ├── assets/      # 資源檔案
│   ├── components/  # Vue 元件
│   ├── App.vue      # 根元件
│   ├── main.js      # 入口檔案
│   └── style.css    # 全域樣式
├── index.html       # HTML 模板
├── package.json     # 專案設定
└── vite.config.js   # Vite 設定
```
