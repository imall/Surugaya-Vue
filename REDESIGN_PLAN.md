# 駿河屋願望清單 - 重構設計計畫

## 🎨 設計理念
- **配色**：淺藍色 + 淺紫色（紫羅蘭）為主，營造清新舒適的視覺體驗
- **排版**：Mobile-First RWD，手機優先設計
- **互動**：平滑過渡、微互動增強用戶體驗
- **一致性**：統一的間距、圓角、陰影系統

## 📱 Mobile-First 策略
- 基礎樣式以手機版為主（寬度 < 640px）
- 使用 `@media (min-width: ...)` 向上擴展
- 避免重複的 CSS 屬性定義

## 🎨 配色使用指南

### Primary（淺藍）
- 主要操作按鈕
- 選中狀態
- 連結

### Secondary（淺紫）
- 次要操作
- 裝飾元素
- 標籤

### Success（綠色）
- 購買完成
- 成功提示
- 庫存充足

### Danger（紅色）
- 刪除操作
- 錯誤提示
- 無庫存

### Neutral（灰色）
- 背景
- 邊框
- 文字
